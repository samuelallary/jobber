what is this?
this is a simple tool to create RSS job search feeds from job portals (ie. LinkedIn), so you can get the latest job offers directly in your RSS reader


get started in four steps
1 - install an RSS reader (see reader recommendations)
2 - create your job search feed here
3 - subscribe to the feed in your RSS reader using the url from step 2
4 - start getting the offers!


reader recommendations

FAQ
why RSS? it's a boring, old, minimalistic technology that does the job
do I really have to install an app? no! you can use a browser extension (see reader recommendations)
which job portals does it use? currently only LinkedIn. you don't need a LinkedIn account to use this service
how often are the feeds refreshed? hourly
can feeds be deleted? yes. if no one uses a feed for a week it will be automatically deleted
is this service free? yes
really? yes
but who's profiting? you!
